# Anti-Duplicator

Anti-duplicator find all occurances of duplicate files and output their relative paths. 

# How to use

```#!bash
$ python3 duplicates.py ./test
These files are the same: ./test/ggl-gk-300x300.png, ./test/anothertest/ggl-gk-300x300.png
```
